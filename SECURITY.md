# Security Policy

## Supported Versions

Only the latest state of `main` is considered supported.

## Reporting a Vulnerability

Do not open a public GitHub issue for sensitive vulnerabilities.

Report security problems privately to **Adriel Zimbril** through:

- Website: `https://www.adrielzimbril.com/`
- GitHub: `https://github.com/adrielzimbril`

Include:

- affected area
- reproduction steps
- impact
- suggested mitigation if known

## Scope Notes

This project depends on external tools and platforms, especially:

- `yt-dlp`
- `ffmpeg`
- Telegram Bot API
- the self-hosted environment used for deployment

When reporting a vulnerability, specify whether it is:

- a PulsorClip application issue
- an environment misconfiguration
- an upstream dependency issue
